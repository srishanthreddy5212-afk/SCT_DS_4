# Task 4: Traffic Accident Data Analysis

## 📌 Objective
Analyze traffic accident data to identify patterns related to weather, road conditions, and time of day.

## 📊 Dataset
- Traffic Accidents Dataset

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Project Description
- Explored accident data.
- Analyzed weather and road conditions.
- Studied accident frequency by time.
- Visualized accident hotspots and trends.

## 🎯 Outcome
Discovered key factors contributing to traffic accidents through data visualization and analysis.
